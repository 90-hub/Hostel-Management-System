# Hostel Management System

## Project Description

This project is a Hostel Management System developed in C++ using Object-Oriented Programming (OOP) concepts. The system helps manage hostel students, room allocation, fee records, and room availability through a menu-driven console application.

## Features

* Add new students
* View student details
* Update student fee records
* Delete student records
* Manage hostel room allocation
* Check room availability
* Fee payment status tracking
* Menu-driven interface

## OOP Concepts Used

* Classes and Objects
* Inheritance
* Encapsulation
* Abstraction
* Polymorphism
* Function Overriding
* Constructors
* Arrays of Objects

## Classes Used

1. Person
2. Student
3. Room
4. Screen
5. StudentScreen

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Console-based Interface

## How to Run

1. Open the project in any C++ compiler.
2. Compile the program.
3. Run the executable file.
4. Use the menu options to manage hostel records.

## Functionalities

### Student Management

* Add students
* View all students
* Delete students
* Update fee records

### Room Management

* Assign rooms
* Free rooms after student deletion
* Display room status

## Conclusion

This Hostel Management System demonstrates the practical implementation of OOP concepts in C++. It provides an easy way to manage hostel students and room records efficiently.
